## Sobre

Guia clínico interativo para diagnóstico diferencial e manejo das arritmias fetais ao ecocardiograma. Desenvolvido para uso educacional por médicos com formação em ecocardiografia fetal.

### Novas funcionalidades — v2.0

- **Taquicardia Juncional Ectópica (JET)** — entidade diagnóstica separada com diferencial de TV e BAVT
- **Síndrome de Costello** — taquicardia atrial multifocal + cardiomiopatia hipertrófica
- **Hidroxicloroquina profilática** — recomendação EULAR 2007 no BAV autoimune
- **Taquicardia sinusal inapropriada** — diagnóstico diferencial com TSV refratária
- **Mnemônicos atualizados** com VA longo diferenciado (TAE vs JET)

### Bases científicas

- Diretriz Brasileira de Cardiologia Fetal – 2019 (Arq Bras Cardiol. 2019;112(5):600-648)
- Lopes LM — Capítulo 16, Ecocardiografia Fetal
- Bravo-Valenzuela NJM — Capítulo 24, Arritmias Fetais
- EULAR 2007/2008 — Hidroxicloroquina no lupus/gestação

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub com o nome `arritmias-fetais`
2. Faça upload dos arquivos `index.html` e `README.md`
3. Em **Settings → Pages → Source**: selecione `main` branch / `/ (root)`
4. O site ficará disponível em `https://seuusuario.github.io/arritmias-fetais/`

## Aviso

Material de uso educacional. Não substitui julgamento clínico individualizado.  
O tratamento intrauterino deve ser realizado exclusivamente em centro terciário com experiência em cardiologia fetal.
